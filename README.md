# metal archives updates

Watches periodically for updates in [Metal Archives](http://www.metal-archives.com/) site and tweet them.

For now, only works with bands updates and additions.

Example:

```
#!/bin/bash
metal-archives-updates \
    -key=$KEY \
    -secret=$SECRET \
    -access_token=$TOKEN \
    -access_secret=$TOKEN_SECRET
    
```
